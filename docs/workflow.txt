PHASE 4: LSTM MODEL DEVELOPMENT & TRAINING
--------------------------------------------------------------------------------

    [Processed Data Ready]
            │
            ├─→ NOTEBOOK 04: Sequence Builder
            │     │
            │     ├─→ Load processed data
            │     ├─→ Define sequence length (e.g., 30 timestamps)
            │     │
            │     ├─→ Create Sequences
            │     │     │
            │     │     ├─→ Sliding Window Approach:
            │     │     │     Input: [t-29, t-28, ..., t-1, t]
            │     │     │     Output: Failure at t+1
            │     │     │
            │     │     └─→ Generate X (features) and y (target)
            │     │
            │     ├─→ Split Data
            │     │     ├── Training Set (70%)
            │     │     ├── Validation Set (15%)
            │     │     └── Test Set (15%)
            │     │
            │     └─→ Save sequence arrays
            │           │
            │           ↓
            ├─→ NOTEBOOK 05: LSTM Model Training
            │     │
            │     ├─→ Import TensorFlow/Keras
            │     │
            │     ├─→ Build LSTM Architecture:
            │     │     │
            │     │     ├─→ Input Layer (sequence_length, features)
            │     │     ├─→ LSTM Layer 1 (128 units, return_sequences=True)
            │     │     ├─→ Dropout (0.2)
            │     │     ├─→ LSTM Layer 2 (64 units)
            │     │     ├─→ Dropout (0.2)
            │     │     ├─→ Dense Layer (32 units, ReLU)
            │     │     └─→ Output Layer (1 unit, Sigmoid)
            │     │
            │     ├─→ Compile Model
            │     │     ├── Optimizer: Adam
            │     │     ├── Loss: Binary Crossentropy
            │     │     └── Metrics: Accuracy, Precision, Recall
            │     │
            │     ├─→ Train Model
            │     │     ├── Epochs: 50-100
            │     │     ├── Batch Size: 32
            │     │     ├── Early Stopping (patience=10)
            │     │     └── Model Checkpoint (save best)
            │     │
            │     ├─→ Evaluate Model
            │     │     ├── Test Set Accuracy
            │     │     ├── Precision & Recall
            │     │     ├── F1-Score
            │     │     ├── ROC-AUC
            │     │     └── Confusion Matrix
            │     │
            │     ├─→ Plot Training History
            │     │     ├── Loss curves
            │     │     └── Accuracy curves
            │     │
            │     └─→ Save: models/motor_lstm_model.h5
            │           │
            │           ↓
            [Trained LSTM Model Ready]


PHASE 5: PREDICTION GENERATION & EXPORT
--------------------------------------------------------------------------------

    [Trained Model Ready]
            │
            ├─→ Develop predict_and_export.py
            │     │
            │     ├─→ Load trained model
            │     │     └── model = load_model('models/motor_lstm_model.h5')
            │     │
            │     ├─→ Load new/test data
            │     │     ├── Read from data/processed/
            │     │     └── Prepare sequences
            │     │
            │     ├─→ Generate Predictions
            │     │     │
            │     │     ├─→ For each motor:
            │     │     │     ├── Create input sequence
            │     │     │     ├── predictions = model.predict(sequence)
            │     │     │     └── failure_probability = predictions[0]
            │     │     │
            │     │     └─→ Compile results
            │     │
            │     ├─→ Add Risk Classification
            │     │     ├── probability > 0.8 → HIGH RISK
            │     │     ├── probability 0.5-0.8 → MEDIUM RISK
            │     │     └── probability < 0.5 → LOW RISK
            │     │
            │     ├─→ Export to CSV
            │     │     └── dashboard/motor_failure_predictions.csv
            │     │           Columns:
            │     │           - motor_id
            │     │           - timestamp
            │     │           - failure_probability
            │     │           - risk_level
            │     │           - recommended_action
            │     │
            │     └─→ (Optional) Upload to Supabase
            │           │
            │           ↓
            [Predictions Generated]
