🔵 PHASE 1 — SUPABASE SETUP (INTEGRATION)

Step 1.1 – Create Supabase Project

Go to Supabase

Create new project

Save:
- Project URL
- Anon key
- Service role key

Step 1.2 – Create Storage Bucket

In Supabase → Storage:

Create bucket: motor-raw-data

Set to private

Upload your motor CSV files here.

Step 1.3 – Create Database Tables

In Supabase SQL Editor:

Login Logs Table
create table login_logs (
  id uuid default gen_random_uuid(),
  user_email text,
  login_time timestamp,
  ip_address text,
  dashboard_accessed text
);


(Optional)

create table model_predictions (
  timestamp timestamp,
  motor_id text,
  failure_probability float
);


✅ Supabase integration complete.
